                          🚀 البدء السريع (5 خطوات)
═══════════════════════════════════════════════════════════════════════════════

1️⃣ الانتقال إلى المجلد:
   cd "d:\Clients projects\dleveria\deliveria"

2️⃣ التثبيت:
   npm install

3️⃣ إنشاء ملف .env:
   MONGODB_URL=mongodb+srv://user:pass@cluster.mongodb.net/deliveria
   JWT_SECRET=your_secret_key
   PORT=8550

4️⃣ التشغيل:
   npm start

5️⃣ الاختبار:
   curl http://localhost:8550/api/orders

═══════════════════════════════════════════════════════════════════════════════
                          💡 نقاط مهمة يجب تذكرها
═══════════════════════════════════════════════════════════════════════════════

1. MONGODB_URL ضروري للعمل الكامل
   → تأكد من الاتصال قبل الإنتاج

2. JWT_SECRET يجب أن يكون آمن
   → لا تستخدم "secret" كقيمة فعلية

3. Firebase مطلوب للإشعارات الفعلية
   → قم بإعداده من Google Cloud Console

4. جميع الراوتر تستخدم checkToken الآن
   → لا تخلط مع verifyToken (تم إصلاح جميعها)

5. State Machine صارم جداً
   → لا يسمح بانتقالات غير صحيحة
